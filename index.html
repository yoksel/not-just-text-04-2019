OpenType

Расширяемый шрифтовой формат

История

  • Разрабатывается совместными усилиями Microsoft и Adobe на основе TrueType и PostScript

Преимущества

  • кроссплатформенный
  • поддерживает Unicode, 65 000+ символов
  • эффективное сжатие
  • улучшенная типографика (лигатуры, дроби, исторические символы) с автозаменой
  • улучшенный кернинг
  • расширяемый формат 🔥
  • EOT — поддерживается только в IE
  • otf — на основе PostScript
  • ttf — на основе TrueType
  • woff — сжатый OpenType шрифт
  • woff2 — более лучше сжатый OpenType шрифт

Существующие форматы

  • SBIX (Apple)
  • CBDT/CBLC (Google)
  • COLR/CPAL (Microsoft)
  • SVG Open Type (Mozilla/Adobe)
SBIX (Apple)
  • содержит растровые изображения
  • много весит (десятки мегабайт)
  • не входит в стандарт OpenType
Поддержка
  • MacOS, iOS, и только в приложениях, которые используют системный рендеринг
  • Chrome, Safari Edge (check it)
CBDT/CBLC (Google)
  • содержит растровые изображения
  • много весит
  • входит в стандарт OpenType
Поддержка
  • Android, Chrome OS
  • Chrome Linux, Firefox, Edge (check it)
COLR/CPAL (Microsoft)
  • глифы в кривых
  • для каждого символа несколько глифов, которые накладываются друг на друга
  • в будущем можно будет перекрашивать через CSS
  • мало весит
  • входит в стандарт OpenType
Поддержка
  • Windows с 8.1 в шрифте Segoe UI Emoji.
  • Браузеры: Firefox, Chrome, Safai, Edge (need to check)
SVG Open Type (Mozilla/Adobe)
  • глифы в SVG, могут содержать градиенты и растроввые изображения
  • в будущем можно будет перекрашивать через CSS
  • мало весит
  • входит в стандарт OpenType
Поддержка
  • Illustrator CC 2018+, Photoshop CC 2017+ и InDesign CC 2019+
  • MacOS Mojave
  • Браузеры: Firefox, Edge (need to check)
Поддержка в браузерах
  • Edge: все форматы
  • Firefox: SVG, COLR, SBIX
  • Chrome: все, кроме SVG Open Type
  • Safari: SBIX, COLR, SVG Open Type (Mojave)
